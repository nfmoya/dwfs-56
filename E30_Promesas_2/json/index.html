Json

JavaScript Object Notation o Notación de Objeto de JavaScript es un formato ligero de intercambio de datos.

Sus características son:

  • Fácil lectura por el humano
  • Ligero de peso
  • Fácilmente “parseable”
  • No permite almacenar funciones
  • Sintaxis similar a la de objetos y arreglos de JavaScript

¿Para qué sirve?

Al momento de transferir información, por ejemplo, de un servidor a un cliente y viceversa, se necesita un formato estandarizado que se pueda entender fácilmente, y sobre todo que sea ligero. Allí es cuando entra en juego el JSON.
